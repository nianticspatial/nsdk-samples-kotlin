# NSDK Samples
This Kotlin project provides examples for many of the Niantic Spatial SDK (NSDK) features.

## __Quick links:__
* [NSDK API Reference](https://www.nianticspatial.com/docs/api/kotlin/?platform=kotlin)
* [Documentation](https://www.nianticspatial.com/docs/nsdk/sample_projects/?platform=kotlin)
* [Niantic Spatial Platform](https://www.nianticspatial.com/docs)

# Getting Started

Clone this repository or download the source code from the Releases tab and open it in Android Studio.

Select the sample or run configuration for the feature you'd like to reference, then build and run on a device or emulator.
If you find yourself having problems using our samples, please refer to our [documentation page](https://www.nianticspatial.com/docs/nsdk/sample_projects/?platform=kotlin) for more information.

To build the samples on an Android device, follow our [Setup Guide](https://www.nianticspatial.com/docs/nsdk/setup/?platform=kotlin).
